# 📊 Profile Dashboard

A modern and responsive **Profile Dashboard** built using **HTML and CSS**. This project displays a user profile, learning progress, skill cards, streak tracker, and a progress button in a clean dashboard layout.

## ✨ Features

* 👋 Welcome Section
* 🎓 Student Profile Information
* 📚 Skill Progress Cards
* 🔥 Current Streak Tracker
* 🚀 View Progress Button
* 🎨 Modern UI Design
* 📱 Responsive Layout
* 💡 Beginner-Friendly Project

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox

## 📂 Sections Included

### Navbar Section

* Welcome Message
* Student Information
* Motivational Quote

### Skills Section

* HTML
* CSS
* JavaScript
* DSA

### Streak Tracker

* Current Learning Streak Display

### Progress Section

* View Progress Button

## 🎯 Learning Outcomes

* Creating layouts using Flexbox
* Working with Classes and Divs
* Using Width and Height effectively
* Applying Margin and Padding
* Styling with Border Radius and Box Shadow
* Building Dashboard Interfaces
